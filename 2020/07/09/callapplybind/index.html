自己实现call，apply，bind

介绍

这三个都是改变函数里的 this 指向
第一个参数都是用来替换 this 的对象，后面是原来函数的参数，不定个数
当第一个参数为 null 或 undefined，则里面 this 指向 window

call，apply用法一样，区别在于

bind 比较特别，返回的是个函数，不执行
